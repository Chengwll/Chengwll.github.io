产品思维

发表于 2019-03-25 | |

1.产品只由用户定义

2.走出办公室找用户聊

3.洞察用户然后创造用户价值

4.用户价值源于用户预期和感知而非客观指标提升

5.在用产品的大都是目标用户，流失的不一定不是

6.产品不需要呵护关心，用户才需要

7.用户是需求的组合

8.需求有层次，要区分挖掘

9.用用户思维才是产品经理，只有平台思维是项目经理

10.好的体验是没有异常

11.好的体验是连贯的

12.好的体验式是用完即走享用还来

13.不要每天都在读驾驶手册，要出车上路

14.快速迭代，频繁复盘

15.做产品经理别做样品经理

16.在不确定下提升事情正确的概率

17.不止选择正确的事情，要选择最正确的事情

18.别太依赖数据

19.数据只提供佐证不提供洞见

20.数据只提供历史不提供预言

21.产品经理提供洞见和预言

22.别单兵作战

23.实现产品需要资源

24.任何人都是资源

25.解决多大的问题，你就有多大的价值

26.像哲学家一样去思考本质

27.常问为什么

28.别和无趣的人共事

29.别给用户你自己不用的产品

30.别为你觉得价值观有问题的公司卖命

31.别给自己设边界

32.别放弃好奇心

33.别做没趣的人

34.做有善意的产品
